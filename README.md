# estudo-catijr
Esse repositório é dedicado ao estudo disponibilzado pela Cati Jr para a realização do projeto trainee.

## Objetivo:
* Repositório dedicado ao acompanhamento do cronograma de estudo para a realização do projeto trainee da Cati Jr.

## Ferramentas utilizadas:
* Python
* Google Colab / Jupyter Notebook
* (conforme for aprendendo novas ferramentas / bibliotecas, irei adicionando)

## Estrutura do repositório:
* **README.md**: Documentação principal do projeto.
* **.gitignore**: Arquivo para ignorar arquivos desnecessários no controle de versão.
* **semana-0n**: Pasta destinada ao conteúdo abordado da semana(n).

## Cronograma:
* Semana 01: Fundamentos de Python para dados, Estrutura de Dados, Funções e Módulos.
* Semana 02: SQL, Numpy, Pandas.
* Semana 03: Matemática para dados.
* Semana 04: Coleta e tratamento de dados.
* Semana 05: Aprendizado de Máquina, Algoritmos Supervisionados.
* Semana 06: Algoritmos não supervisionados.
* Semana 07: Streamlit.
* Semana 08: Apresentação.
